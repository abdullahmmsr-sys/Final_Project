1. **تقسيم حسب الضوابط**: كل ضابط يمكن أن يكون embedding منفصل
2. **تضمين السياق**: إضافة اسم المجال والمجال الفرعي لكل ضابط
3. **البيانات الوصفية**: استخدام IDs والمجالات كـ metadata

```json
}
```
